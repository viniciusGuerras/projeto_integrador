## Como rodar o projeto localmente com Vite

1. **Pré-requisitos**:  
   - Node.js (recomenda-se a versão LTS)  
   - npm ou yarn

2. **Instalar as dependências**:  
    No terminal, dentro da pasta do projeto, execute:

    ```bash
    npm install
    ou
    yarn

3. **Rodar o projeto localmente**:
    npm run dev
    ou
    yarn dev
